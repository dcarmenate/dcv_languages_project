Rust

Rust “ is an empowering programming language empowering everyone to build reliable and efficient software”(first source). Rust was created by Graydon Hoare while he was working for Mozilla, the open-source company. He created the language in hopes that he could write small but fast code without memory bugs and released it to the public in 2010. Companies like Microsoft and Amazon regard it as a key to future coding. Rust is memory efficient and can perform critical services and run on embedded devices.

What makes Rust special?

  • Concurrency- Built a support concurrent through its programming system that enforces strict rules for data accessin.
  • No garbage collector- Rust does not employ garbage collection.
  • Type of inference- Rust compiler automatically detects an expression based on context while you code.
