The programming language PHP was created in 1994 by Rasmus Lerdorf. The language was created to track the visits to his online resume. Over time rasmus added more functionality to produce a larger implementation. PHP has evolved from a simple web builder into a scripting language for web applications. Today the programming language is used for operating websites validating html forms and executing code loops. The programming language is still used by web developers; Php is still one of the highest programming languages used in countries like France at 43% and Northern Europe at 33%.
